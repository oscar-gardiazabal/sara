Add Categories Post Footer automatically add any custom paragraph.

== Description ==

Add Categories Post Footer Plugin is very flexible and easy to use. The global options can be customized via wordpress admin  option panel. Moreover the global options can override by adding special custom field to each post you want to customize its  configuration individually. This will allow you to disable the appearance of some option or even add special code or paragraph for  particular post. Please refer to instruction below in this page for detailed instruction.


== Installation ==

   1. Unpack file package using your favorite zip software.
   2. Upload .php file to your wordpress plugin directory on your server.
   3. Login to wordpress admin and go to plugin tab and activate Add Categories  Post Footer Plugin.
   4. Go to Option => Add Categories  Post Footer Categories tab to config the plugin option.

For SimpleTags user: You have to disable the original SimpleTags Plugin in order to show Technorati tags at the nd of the post.  Otherwise the tag list will appear before Add footer section.

== Upgrade ==

Upgrade Add Categories Post Footer is easy, just replace plugin file (add_categories_post_footer.php) in plugin directory with  new version. No need to re-config any options in "Post Footer" option tab.
